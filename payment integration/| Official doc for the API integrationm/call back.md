# Payment Callback URLs

Վճարային համակարգերի callback URL-ներ borboraqua.am նախագծի համար։

---

## Ameriabank

| Պարամետր | URL |
|----------|-----|
| **Success** | `https://borboraqua.am/wc-api/ameriabank_successful` |
| **Failed** | `https://borboraqua.am/wc-api/ameriabank_failed` |
